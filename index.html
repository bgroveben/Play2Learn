    Math Facts

    This is a simple game for practicing addition, subtraction, multiplication, and division. The problem appears with a text field below it. Enter values using the keyboard or the buttons on the screen, and see how many correct answers you can get in 30 seconds.

    Anagram Hunt

    An anagram is a word made by rearranging the letters of another word. For example, post, pots, opts, tops, and spot are all anagrams of stop. The object of the Anagram Hunt game is to find all the anagrams for a given word.
